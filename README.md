### Installation

You can get Started with Portfolio in 3 Simple Steps

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/ashifa454/portfolio
$ cd portfolio
$ npm install
$ npm start
```

For Deploying on Firebase ...

```sh
$ npm run build
$ firebase init
$ firebase deploy
```
